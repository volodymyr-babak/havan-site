# Гавань — котеджне містечко (новий сайт)

Statyc redesign of `havan.com.ua`. No build step, no dependencies — plain HTML/CSS/JS.

## Структура

```
index.html              # одна сторінка, секції через плавний скрол
assets/css/styles.css   # стилі (дизайн-токени вгорі файлу)
assets/js/main.js       # хедер, мобільне меню, анімації, FAQ, форма
assets/img/             # зображення
```

## Локальний перегляд

```bash
cd /tmp/havan-site
python3 -m http.server 8080
# відкрити http://localhost:8080
```

## Деплой в інтернет

Це статичні файли — підходить будь-який хостинг:
- **Netlify / Vercel / Cloudflare Pages** — перетягнути теку `havan-site`.
- **GitHub Pages** — запушити вміст у репозиторій, увімкнути Pages.
- **nginx / Apache** — скопіювати теку в корінь сайту.

## Що варто оновити реальними даними

Зараз більшість фото — тематичні стокові зображення (вода, будинки, природа),
крім `assets/img/havan-real.jpeg` — це справжнє фото містечка.

1. **Фото** — замінити файли в `assets/img/` на справжні (зберегти ті самі імена,
   щоб не правити HTML). Головне фото: `hero-water.jpg`.
2. **Контакти** — додати телефон та адресу в секції «Контакти» (`index.html`,
   блок `.cinfo`).
3. **Документи** — оновити посилання на актуальний PDF договору та додати тарифи
   (`index.html`, секція `#docs`).
4. **Новини** — додати свіжі записи в секцію `#news`.

## Дизайн

- Шрифти: **Cormorant Garamond** (заголовки) + **Manrope** (текст), Google Fonts.
- Палітра, відступи й кольори — CSS-змінні у `:root` (початок `styles.css`).
