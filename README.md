# Гавань — котеджне містечко (новий сайт)

Статичний маркетинговий сайт котеджного містечка «Гавань» на річці Десна.
Без збірки та залежностей — чистий HTML / CSS / JS.

**Live:** https://volodymyr-babak.github.io/havan-site/

## Структура

```
index.html              # односторінковий сайт, секції через плавний скрол
assets/css/styles.css    # стилі (дизайн-токени у :root на початку файлу)
assets/js/main.js        # хедер, мобільне меню, scroll-анімації, FAQ, форма
assets/img/              # зображення (фото, AI-візуалізації, генплан, favicon)
assets/docs/             # PDF-документи (договори про надання послуг)
```

### Секції сторінки
1. **Hero** — головний банер
2. **Про містечко** — опис + реальне аерофото
3. **Перспектива (Vision)** — AI-візуалізації майбутнього вигляду
4. **Ділянки / Генеральний план** — карта ділянок + легенда + CTA
5. **Послуги** — інфраструктура та сервіс (6 карток)
6. **Галерея** — реальні аерофото
7. **Новини**
8. **Документи** — договори (PDF) + генплан
9. **FAQ**
10. **Контакти** — телефон, пошта, форма


## Деплой

Сайт розгорнуто на **GitHub Pages** з гілки `main` (репозиторій
`volodymyr-babak/havan-site`). Будь-який push у `main` автоматично оновлює сайт
за ~1 хвилину:

```bash
git add -A && git commit -m "опис змін" && git push
```

Альтернативи (статика працює всюди): Netlify, Cloudflare Pages, Vercel, nginx.

## Як оновити контент

1. **Фото** — замінити файли в `assets/img/` (зберегти ті самі імена, щоб не
   правити HTML).
2. **Нові AI-візуалізації** — додати `street.jpg` (в’їзд/КПП) і `plot.jpg`
   (вільна ділянка) у секції Vision / Ділянки.
3. **Документи** — покласти новий PDF у `assets/docs/` і оновити посилання в
   секції `#docs`.
4. **Ділянки** — оновити легенду/опис у секції `#plots`; нову версію генплану
   зберегти як `assets/img/plan.jpg`.
5. **Новини** — додати картки в секцію `#news`.

## Дизайн

- Шрифти: **Cormorant Garamond** (заголовки) + **Manrope** (текст), Google Fonts.
- Тема: темний charcoal-teal фон, бронзово-золотий акцент.
- Палітра, відступи й кольори — CSS-змінні у `:root` (початок `styles.css`).
